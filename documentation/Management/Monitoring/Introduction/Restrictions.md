# 限制说明

云监控的各个功能的配额限制如下表所示：

| 限制项                               | 限制规则 |
| ------------------------------------ | -------- |
| Dashboard中可创建的监控面板数量      | 20个     |
| 单个监控面板中的监控图表数量         | 20个     |
| 单个监控图表中的监控项               | 10个     |
| 自定义监控命名空间数量               | 20个     |
| 报警规则中单条规则关联指定实例的数量 | 100个    |
| 报警规则中单条规则出发条件数量       | 10个     |
