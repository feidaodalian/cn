# 操作流程

介绍购买并使用云原生实时数仓 Starwift 集群的完整过程，旨在帮助新用户快速上手并正确使用 Starwift 。

## 注意事项
- 如果创建计费方式为按配置的资源，请确保账户余额（或代金券金额）不少于 50 元。
- 目前仅支持华北-北京地域。
- 如果为子账号，请确保已经开通 Starwift 的创建，查看等相应的权限 。
-
