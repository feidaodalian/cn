# 实时性能
实时展示数据库实例各项性能指标。

## 前置条件

* 已创建数据库实例，数据库类型目前支持MySQL、Percona
* 数据库实例已接入SmartDBA服务

## 操作序列
控制台->数据库与缓存->SmartDBA->实例监控->实例监控详情->实时性能。

## 查看实时性能
默认监控指标全选并按序展示，默认开启图表联动。
