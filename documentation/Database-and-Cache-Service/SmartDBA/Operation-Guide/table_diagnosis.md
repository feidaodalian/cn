# 表诊断

对表的常见问题进行分类展示，并对具体问题提供优化建议。

## 前置条件

* 已创建数据库实例，数据库类型目前支持MySQL、Percona
* 数据库实例已接入SmartDBA服务

## 操作步骤

1. 登陆京东云控制台，进入数据库自治服务SmartDBA管理页面；
2. 左侧导航栏选择**实例监控**，进入实例监控列表页；
3. 找到目标实例，点击【性能监控】进入实例监控详情页面；
4. 选择**表诊断**Tab，进入诊断页面查看问题和诊断建议；
