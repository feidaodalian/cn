# 系统设置-通知

### 功能说明

系统设置中的通知功能，可以设置当出现告警威胁攻击时，选择时间进行短信和邮件的及时响应与通知。

- 通知列表

包含托管pin/子账号、被托管pin/子账号、通知设置、纳管资源、状态、操作等，其中通知设置与通知接收人需要安全运营人员进行设置。

![](../../../../../image/CSoC/CSoC-07-1-1.png)

- 通知设置

自定义配置通知项，通知等级、通知方式及时间。
