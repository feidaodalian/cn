## 产品功能
### 功能概述
#### 全面的文件管理
- 根据组织机构及权限统一管理企业文件;
- 支持常见文件类型在线预览;
- 支持文件添加标签并通过标签搜索;
- 支持不限上限历史版本，提供版本还原。
#### 无间的协作共享
- 支持文件在线发送、文字沟通、及时触达；
- 部门内文件可发表评论意见；
- 对外分享可设置密码与有效期，保证文件安全。
#### 懂企业的工作流设计
- 从企业工作流视角出发，融入多项团队协作能力：
- 一对一文件投递、文档讨论、多人在线编辑、外发审批、文件双向同步、及时消息通知等。
#### 专属的企业文档库
为企业打造的高效知识管理平台，能够为企业轻松搭建一个知识管理库，操作简单便捷，管理员添加成员后即可实现多人共享、实时同步，团队成员可以在其中记录内部协作产品的所有内容。<br>
  - 公共文档库：存放公共资源；
  - 项目文档库：存放独立的项目资料。
