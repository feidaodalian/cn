# 分类分级策略

## 敏感数据分级

您可以根据自身公司的安全要求对数据进行分级的定义；当前数据安全中心支持4个数据等级，如下表：

敏感分级 | 敏感等级 | 描述
---------|----------|---------
 L1 | 0 | 公开数据
 L2 | 1 | 秘密数据
 L3 | 2 | 机密数据
 L4 | 3 | 绝密数据

其中`敏感分级`及`描述`可以根据您公司的安全需求进行自定义的调整。

在控制台调整敏感数据分级的路径为： 云安全->数据安全中心->敏感数据管理->敏感数据分级
