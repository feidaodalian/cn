# 验证配置生效

此步骤 **不是必须步骤**  
<Br/>规则设置完成后，开启全局开关和规则开关，配置生效需要一段时间，可以通过以下步骤验证规则是否生效。

## 测试步骤

例如配置一条禁止本地IP访问云主机80端口的规则，使用本地命令行，telnet访问云主机公网IP 80端口，结果显示连接失败，证明规则已生效。
