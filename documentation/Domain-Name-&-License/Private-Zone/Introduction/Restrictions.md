# 限制说明

## 规格列表

| 规格列表 | 限制数量 | 限制说明 |  
| :------ | :---------: | :---------: |
| 域名    | 50个 | 每个套餐最多支持购买50个域名 |
| 关联VPC  | 30个       | 每个域名最多支持关联30个VPC  |
| 解析记录  | 10万条      | 每个域名下最多支持添加10万条解析记录  |  
| 子域名级别  | 20级      | 每个域名下最多支持添加20级子域名解析记录  |  
| 单个VPC每秒请求  | 5000次/秒    | 单个VPC每秒请求超过5000次，将被限速，无法保证99.99%的SLA  |
