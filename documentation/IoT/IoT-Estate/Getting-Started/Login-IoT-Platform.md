# 登录物联平台

1.登录智能生活物联网平台服务控制台, 在左侧导航栏点击**服务详情**

2.在**服务详情**页，点击平台地址即可跳转至智能生活物联网平台

![用户登录](../../../../image/IoT/IoT-Estate/Getting-Started/Service-Detail.png)3.使用服务列表所展示的账号和初始密码，登录智能生活物联网平台
